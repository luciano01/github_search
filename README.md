## For developers
> To clone locally
```
$ git clone git@github.com:luciano01/github_search.git
$ cd github_search
```
> Open the project in an editor, if you use VS Code, follow this
```
$ cd github_search
$ code .
```

> To run, follow this on terminal
```
$ flutter clean
$ flutter pub get
$ flutter run --release
```
